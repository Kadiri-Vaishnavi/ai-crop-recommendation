🌾 AI Crop Recommendation System

Welcome to the AI Crop Recommendation System! This project helps farmers choose the best crops for their soil and environmental conditions using AI & Machine Learning.

🌐 Live Demo

Check out the live version here: AI Crop Recommendation

🖼️ Features
✅ User authentication: Login and registration
✅ Crop recommendation based on soil and environment data
✅ Responsive frontend interface with HTML, CSS, and JavaScript
✅ Machine Learning backend predicts optimal crops

🛠️ Tech Stack
HTML5 – Structure of the frontend
CSS3 – Styling and responsive design
JavaScript – Frontend interactivity
Node.js / Python – Backend server for login and ML predictions

📁 Folder Structure

📌 How to Use
Clone the repository:
git clone https://github.com/Kadiri-Vaishnavi/ai-crop-recommendation.git
Navigate to the project folder:
cd ai-crop-recommendation
Frontend: Open index.html in your browser to view the site locally.
Backend: Deploy the server folder to a cloud platform (Render, Railway, or Heroku) to enable login, registration, and ML predictions.
✨ Notes
GitHub Pages only serves static files — login/register won’t work without a running backend.
Ensure all file paths (CSS/JS) are correct and case-sensitive.
